# jax-for-numpy-users
Materials for "Effortless Hardware Acceleration and More with JAX", a workshop given on behalf of Princeton University Research Computing in March 2026

The material in the notebook and slides relies heavily on the [JAX documentation](https://docs.jax.dev/en/latest/), especially for worked examples.

## How to use

I recommend using the link provided to run the example notebook in [Google colab](https://colab.research.google.com/).
If you are using your own `Python` environment, the only requirements are `jax` and `numpy`.
Installation instructions for [jax](https://docs.jax.dev/en/latest/installation.html) and [numpy](https://numpy.org/install/).
